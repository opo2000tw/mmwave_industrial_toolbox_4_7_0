TI mmWave Certification Database Release Notes =================================== Overview ======== TI mmWave Certification Dataase is a repository of the certification documents, test reports and other collaterals relating to various requlatory compliance certification done on the mmWave radar evaluation modules. TI mmWave Radar Device Regulatory Compliance Guide appnote is available on TI.com to provide some technical insights on the test standards.
