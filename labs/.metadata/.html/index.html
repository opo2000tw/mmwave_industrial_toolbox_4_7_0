Labs
The labs are source code examples on how to get started with the mmWave sensors. Each lab provides detailed instructions to follow along while evaluating. Source code of each lab can be loaded directly into Code Composer Studio through the integrated view in TI Resource Explorer.

Chirp Database

The chirp database is an easy-to-search repository of different mmWave RF front-end configurations that the TI mmWave team has calculated for our demos and experiments. These chirp designs are available for you to get started designing quickly and can be loaded directly into mmWave Sensing Estimator.

Antenna Database

13
Antennas
mmWave Antenna Database is a repository of mmWave antennas put together by the TI mmWave team and third-parties. TI-designed antennas are available to get started by reusing the stack-up and layout to use the same antenna for your application. Each third-party designed antenna can be used as an antenna module available from that third-party to order and use. Each antenna design is validated as a part of a board design.

How to access previous Industrial Toolbox versions

From Code Composer Studio:

Open the TI Resource Explorer, click on the Home icon, select the desired version for the Industrial Toolbox and click the Download button

From the web:

Go to the mmWave Industrial Toolbox, in the feature menu choose Change Version then select the desired version for the Industrial Toolbox.
