Hardware Setup of MMWAVEICBOOST + Antenna Module for Functional Mode =========== This guide is intended for users with an **MMWAVEICBOOST carrier board** and **antenna module board**. The guide will walk through setting up the boards into **Functional Mode**. In functional mode, the EVM boots autonomously from flash and starts running the bin image. <img src="images/carrier_and_antenna_module/carrier_modules.png" width="600"/> # 0. Requirements ### Required and Supported mmWave Evaluation Modules (EVM) Quantity | Item ------------------|----------------- 1 | [MMWAVEICBOOST Carrier Board](http://www.ti.com/tool/MMWAVEICBOOST) 1 | Antenna Module Board: [IWR6843ISK](http://www.ti.com/tool/IWR6843ISK), [IWR6843ODS](http://www.ti.com/tool/IWR6843ISK-ODS), or [IWR6843AOPEVM](http://www.ti.com/tool/IWR6843AOPEVM) ### Additional Requirements Quantity | Item | Details ---------|--------------------------|----------------- 1 | Micro USB Cable | Provided with the MMWAVEICBOOST 1 | Power Supply | 5V, 3A with 2.1-mm barrel jack (center positive). The power supply can be wall adapter style or a battery pack with a USB to barrel jack cable. {{b Steps 1-4 may be skipped if the hardware setup has not change since following the guide for [Hardware Setup of MMWAVEICBOOST + Antenna Module for Flashing Mode](../hardware_setup/hw_setup_antenna_module_and_carrier_for_flashing.html) }} # 1. Configure Antenna Module This step only applies if using the AOPEVM antenna module. Users of the ISK or ODS antenna modules can skip to the next step. To use the AOPEVM as an antenna module in conjunction with the ICBOOST carrier card the switches need to be set as shown to route signals from the carrier card through the 60Pin header. ### Front of AOPEVM <img src="images/carrier_and_antenna_module/aop_front_carrier_mode.png" width="600"/> ### Back of AOPEVM <img src="images/carrier_and_antenna_module/aop_back_carrier_mode.png" width="600"/> # 2. Mount Antenna Module Mount the selected antenna module on the 60Pin HD connectors. * If using the ISK or ODS module, both 60Pin connectors will be used. * If using the AOPEVM module, only the right connector closer to the micro-usb connectors will be connected. <img src="images/carrier_and_antenna_module/mount_antenna_modules_on_carrier.png" width="500"/> The mounted modules should appear as shown below. <img src="images/carrier_and_antenna_module/mounted_modules.png" width="600"/> # 3. Set MUX Controls on MMWAVEICBOOST for Standalone Operation Set the S1 switch combination as shown below. This puts the MMWAVEICBOOST in standalone operation as opposed to in development mode for use with a DCA1000. <img src="images/carrier_and_antenna_module/carrier_mux_control_standalone.png" width="500"/> # 4. Connect micro-USB and Power * Connect the micro-USB to the carrier board at the connector shown and to the PC. Note that the micro-USB is connected to the lower XDS110 USB (and not the FTDI USB). * Connect the 5V/3A power supply to the power source and then to the carrier board. LEDs on the carrier board should turn on. <img src="images/carrier_and_antenna_module/carrier_connect_cables.png" width="400"/> # 5. Set SOP Mode for Functional Mode Enable functional mode by setting SOP2:0 to 001 as shown below. <img src="images/carrier_and_antenna_module/carrier_sop_mode_functional.png" width="500"/> # 6. Toggle NRST switch To ensure that the correct SOP mode is latched once power has been applied, power cycle by pressing and releasing the NRST switch identified below. The yellow DS1 LED below the switch should toggle OFF and ON when the switch is depressed then released. <img src="images/carrier_and_antenna_module/carrier_assert_nrst.png" width="400"/> [[y!
