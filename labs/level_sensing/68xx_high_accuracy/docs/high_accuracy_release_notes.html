Version 3.2.0 ============= Whats New? ======================== * This 68xx verion of this lab now includes a multi-peak detection detection feature which performs the Zoom FFT on three distinct objects in the scene. * The [High Accuracy Visualizer](https://dev.ti.com/gallery/view/4768107/High_Accuracy_Visualizer/ver/2.0.0/) has been updated with color indicators and also compatibility for the 68xx Multi-Peak Version. Overview ======== This lab demonstrates the use of TI mmWave sensors for applications requiring a high degree of range accuracy. Using the Zoom-FFT technique one can achieve range accuracy on the order of millimeters. * Supported EVM platforms: * IWR1443BOOST (Compliant with SDK 2.1) * IWR1642BOOST (Compliant with SDK 2.1) * IWR6843ISK (ES2.0) (Compliant with SDK 3.3) ** (Both 68xx and 64xx versions) ** * IWR6843ODS (ES2.0) (Compliant with SDK 3.3) ** (Both 68xx and 64xx versions) ** * IWR6843AOP (Compliant with SDK 3.2 AOP) ** (Both 68xx and 64xx versions) ** Features ======================== * mmWave SDK 3.x compliant for IWR6843 Versions * mmWave SDK 2.1 compliant for IWR1443/1642 Versions ---------------------------------
