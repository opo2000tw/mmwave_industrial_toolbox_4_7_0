Hardware Setup of IWR6843ISK/ODS for Flashing Mode =========== This guide is intended for users with an **IWR6843ISK and IWR6843ISK-ODS** (EVM). The guide will walk through setting up the board into **Flashing Mode** as a standalone EVM. NOTE: The [MMWAVEICBOOST Carrier Board](http://www.ti.com/tool/MMWAVEICBOOST) is **NOT** necessary to flash the IWR6843ISK Rev.C and IWR6843ISK-ODS Rev.B. If desired, these modules can be used as an antenna module board with the MMWAVEICBOOST.
